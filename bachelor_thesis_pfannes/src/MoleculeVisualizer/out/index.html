Installation

You need to make sure you have the latest version of node.js installed.
The main component of the project is a visualization library called three.js. All libraries used by the project are managed via the npm package manager. Install them by executing the following commands:

> cd bachelor_thesis/src/MoleculeVisualizer
> npm install

Usage

MoleculeVisualizer is built as a single page application, so you need to run a local webserver in order to use it. The simplest solution is to use the serve package from npm by running the following inside the MoleculeVisualizer directory:

> npx serve

After executing this command, copy the URL that shows up in the CLI into your browser and the MoleculeVisualizer should start up.
